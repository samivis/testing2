# Rec Engine

Recommends one movie based on your mood and how much time you have.

**The idea worth stealing:** collect a couple of details from the user, then ask the AI to commit to ONE specific pick (not a vague list). This pattern works for almost any "what should I ___?" tool: what to watch, read, cook, listen to, or wear.

## Run it
```
python3 examples/rec-engine/main.py
```

## Make it your own
- Swap movies for books, recipes, songs, or workouts.
- Add more inputs (budget, genre, who you're with).
- Turn it into a web app with dropdowns or text boxes for each input.
