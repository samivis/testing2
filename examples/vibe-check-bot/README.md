# Vibe Check Bot

Reads a message you're about to send and tells you the vibe, a score, and one tweak to make it better.

**The idea worth stealing:** AI doesn't just *write* things — it can *judge* and *explain* things. Asking for a rating + a reason + a suggestion is a powerful pattern you can reuse for tons of tools (rate my essay, rate my outfit, rate my bio).

## Run it
```
python3 examples/vibe-check-bot/main.py
```

## Make it your own
- Point it at a different kind of input: an Instagram caption, an email to a teacher, a bio.
- Change the scale (1–5 stars instead of 1–10).
- Turn it into a web app where users paste text and get the vibe back.
